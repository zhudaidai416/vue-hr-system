# vue-hr-system

## 介绍

基于 vue2+js+vuex 的人力资源后台管理系统

基于 vue-admin-template 架构进行二次加工

## 环境依赖

```bash
npm install
```

## 运行

```bash
npm run dev
yarn dev
```
